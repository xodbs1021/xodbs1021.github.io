소프트웨어 마에스트로가 시작됐다. 팀은 3인, 프로젝트는 AI 기반 라이브 스트리밍 하이라이트 자동 클립 추출 시스템이다. 내 역할은 미디어 도메인 백엔드 — FFmpeg, HLS, ABR, CDN.

이 글은 6개월 후 이력서에 있을 것들을 미리 써두는 글이다. 증명은 나중에 한다.

---

## 이력서 목표 5줄

**1. HLS 세그먼트 경계·IDR 프레임 처리로 '프레임 정확' 클립 추출 구현 — 잘린 프레임 발생률 0.3% 이하 유지**

**2. FFmpeg + NVENC로 1080/720/480 ABR 래더 자동화, 트랜스코딩 단가 기준 대비 60% 절감**

**3. S3 multipart + GPU 트랜스코딩 병렬 처리로 라이브 종료 → 클립 발행 end-to-end p95 90초 달성**

**4. CMAF 패키징 + hls.js 통합으로 라이브·VOD 메타데이터 정합 함정 5가지 해결**

**5. 치지직·SOOP 공식 API 봇·자동화 정책 준수 운영 — OAuth 토큰 관리, 영상 데이터 보유·삭제 정책 1쪽 운영**
