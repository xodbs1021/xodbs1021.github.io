## 레벨 3-5 / 3강: 코덱 라이센스 — 비용과 법적 부담

---

### 라이센스를 왜 알아야 하나

개발자가 코덱 결정할 때 기술만큼 중요한 요소.

```
[라이센스가 비즈니스에 미치는 영향]
1. 디바이스 출하 시 비용 (제조사)
2. 콘텐츠 배포 시 비용 (스트리밍)
3. 인코딩/디코딩 소프트웨어 비용
4. 법적 리스크 (특허 분쟁)
```

기술자 답변에 라이센스 빼면 면접관이 깊이 부족하다고 본다.

---

### MPEG LA — H.264 라이센스 풀

H.264 라이센스를 관리하는 회사.

```
[MPEG LA의 H.264 풀]
참여 특허권자: 약 30개 회사
- Microsoft, Apple, Samsung, Sony, LG 등

라이센스 비용 (2024년):
- 디바이스: 처음 10만대 무료, 이후 $0.20/대
- 콘텐츠: 12개월 무료 시청자 분, 이후 비용
- 연간 최대 한도: 약 $9.75M
```

H.264가 사실상 표준이 된 이유 = MPEG LA의 합리적 라이센스.

```
[디바이스 제조사 부담]
삼성 갤럭시 연 1억대:
- H.264 라이센스: $19.5M/년
부담되지만 받아들일만함
```

---

### H.265의 라이센스 지옥

H.265는 라이센스가 여러 풀로 분산됨.

```
[H.265 라이센스 풀]
1. MPEG LA (일부 특허)
   - 디바이스: $0.20/대
   - 연 한도: $25M

2. HEVC Advance (다른 특허)
   - 디바이스: $0.20-$1.50/대
   - 콘텐츠: $0.05/구독자/월
   - 연 한도: $40M

3. Velos Media (또 다른 특허)
   - 비용 비공개
   - 협상 필요

4. 개별 특허권자
   - Technicolor (직접 라이센싱)
   - 다른 회사들
```

여러 곳에 비용 지불. 협상 부담.

```
[삼성 갤럭시 연 1억대]
- MPEG LA: $20M
- HEVC Advance: $20-150M
- Velos: 협상 결과
- 총: 수천만~수억 달러
```

H.264의 5~10배 비용. 이게 보급 장벽.

---

### Apple의 H.265 전략

Apple은 H.265를 적극 채택.

```
[Apple의 H.265 결정]
2014년 iPhone 6에 H.265 디코더 내장
2017년 iPhone 7+ HEIF (이미지), HEVC (영상)
4K 녹화 H.265 강제

이유:
- 자사가 HEVC Advance 일부 특허 보유
- 즉, 자기들이 라이센스 수입 받음
- 4K 시대 대비 효율 필수
```

Apple은 라이센스 비용보다 라이센스 수입이 큰 입장.

이게 다른 회사와 인센티브 차이.

---

### VP9, AV1의 무료 모델

```
[AOMedia의 라이센스 모델]
멤버 회사: 자기 특허를 무료로 기여
서로 특허 침해 클레임 금지

→ AV1 사용 시 라이센스 비용 0
→ 단, AOMedia 멤버 가입 시 자기 특허 기여 의무
```

이게 거대 기업들이 AOMedia에 가입한 이유:
- 라이센스 수입 없음 (자기 특허 무료 기여)
- 대신 다른 멤버 특허 무료 사용
- 결과: 모두 0원

---

### Sisvel — AV1 특허 위협 시도

2019년 AOMedia에 도전.

```
[Sisvel V-Codec Patent Pool]
- 특허 풀 운영 회사
- "AV1에도 우리 특허가 있다" 주장
- 라이센스 비용 요구

AOMedia 대응:
- 법적 검토
- 특허 회피 설계
- 강한 거부
```

법적 분쟁 진행 중이지만 AOMedia가 우세.

대형 기업 + 명확한 무료 정책 = Sisvel 약화.

---

### 실무에서 라이센스 부담

작은 회사 입장.

```
[스트리밍 스타트업 - 100만 시청자]

H.264 사용:
- 라이센스 비용: 거의 0 (작은 규모는 면제)
- 안전

H.265 사용:
- 라이센스 비용: 협상 필요
- 협상 안 하면 법적 리스크
- 콘텐츠당 비용

AV1 사용:
- 라이센스 비용: 0
- 다만 인코딩 비용 큼
```

작은 회사는 H.264 또는 AV1. H.265 도입 시 변호사 + 협상 비용 부담.

---

### 한국 회사의 코덱 라이센스 부담

```
[삼성, LG, 현대]
H.264 라이센스: 지불 (디바이스 출하)
H.265 라이센스: 지불 + 일부 풀에 참여 (자기 특허 기여)
AV1: 무료 + 자기 특허 일부 기여

[중소 스트리밍 회사]
H.264: 매출 작으면 무료
H.265: 협상 부담
AV1: 사용 시 무료
```

치지직 같은 회사도 H.265 도입 시 라이센스 비용 협상 필요.

이게 H.264 유지하는 또 다른 이유.

---

### 오픈소스 인코더의 라이센스

x264, x265, libvpx, libaom 같은 소프트웨어.

```
[오픈소스 인코더]
x264: GPL 라이센스 (소스 공개 의무)
x265: GPL + 상용 라이센스 (선택)
libvpx: BSD (자유)
libaom: BSD (자유)
SVT-AV1: BSD (자유)
```

GPL인 x264, x265는 자기 코드도 공개해야 함.

```
[FFmpeg 빌드 시 라이센스]
LGPL 빌드 (기본): libavcodec 등만 사용 가능
GPL 빌드: x264, x265 포함 가능
non-free 빌드: libfdk_aac 포함 가능
```

FFmpeg 다운로드 시 어떤 빌드인지 확인 필요.

### Patent vs Codec Patent — 별개 이슈

코덱과 별도로 다른 특허 위험.

```
[추가 위협 사례]
H.264 콘텐츠 인코딩:
- MPEG LA 라이센스 외에도
- 개별 특허 침해 클레임 가능
- (드물지만 발생)

음악 코덱:
- AAC: Fraunhofer 등
- MP3: 2017년 만료 (이제 자유)
```

기업이 코덱 사용 시 변호사 검토. 작은 프로젝트는 부담 없음.

### YouTube, Netflix의 라이센스 전략

대규모 OTT가 다루는 방식.

```
[Netflix]
H.264: MPEG LA + 협상
H.265: HEVC Advance 등과 협상
VP9: 무료
AV1: 무료 (AOMedia 멤버)

다양한 코덱 사용해 협상 우위:
"H.265 비용 안 내리면 AV1 전환"
```

코덱 다양화 = 협상 카드.

### 라이센스 검토 - 실무 흐름

새 코덱 도입 결정 시.

```
[검토 단계]
1. 기술 평가 (압축률, 호환성, 인코딩 속도)
2. 라이센스 확인
   - 어떤 풀에 라이센스 있나
   - 디바이스 비용
   - 콘텐츠 비용
3. 협상 또는 가입
   - HEVC Advance 직접 협상
   - AOMedia 멤버십
4. 법적 검토 (변호사)
5. 최종 결정
```

대형 회사는 코덱 결정에 수개월 소요.

### 면접에서 라이센스 답변

"왜 H.265 안 도입하셨어요?" 질문.

```
"기술적으로 H.265는 우수하지만 라이센스 부담이 큽니다.

H.265는 MPEG LA, HEVC Advance, Velos Media 등 여러 풀에 라이센스 비용을 내야 합니다.
저희 규모에서 정확한 비용은 협상 사안이지만,
H.264 대비 5~10배 부담이라고 알려져 있습니다.

저희 시청자가 모바일 위주고 H.265 효율 절감 효과가 큰 4K 콘텐츠 비중이 낮아
라이센스 비용보다 절감 효과가 작을 가능성이 큽니다.

대안으로 AV1을 검토하고 있습니다. AV1은 라이센스 무료고 H.265보다 효율도 좋습니다.
다만 시청자 측 보급률이 낮아 H.264와 병행 운영 필요한데, 그 시점을 보고 있습니다."
```

기술 + 비용 + 시장 분석 답변. 강한 인상.

### 코덱 vs 컨테이너 라이센스

종종 헷갈리는 부분.

```
[코덱 라이센스]
H.264, H.265, AAC: 비용
VP9, AV1, Opus, MP3: 무료

[컨테이너 라이센스]
MP4, MPEG-TS: 무료
WebM: 무료
FLV: Adobe 보유 (사실상 무료 사용)
```

컨테이너는 거의 무료. 코덱이 비용 부담.

### 미래 - 라이센스 분쟁 전망

코덱 라이센스 시장 변화.

```
[2030년 예상]
H.264: 일부 특허 만료, 비용 감소
H.265: 점진적 사용 감소 (AV1 대체)
VP9: AV1로 흡수
AV1: 무료 + Sisvel 같은 도전자
VVC (H.266): 라이센스 복잡, 보급 더딤
EVC: Baseline 무료로 일부 채택
```

무료 코덱 (AV1) 진영이 점점 강해질 것.

라이센스 비용이 큰 H.265는 4K 영화 등 특수 영역만 유지.

---

### 3강 핵심 3줄

1. H.264는 MPEG LA 한 곳 라이센스로 단순, H.265는 MPEG LA + HEVC Advance + Velos Media + 개별 특허로 복잡 — 부담 5~10배
2. AOMedia는 멤버 회사가 자기 특허를 무료 기여하는 모델 → AV1 사용 시 라이센스 0원, Sisvel 같은 도전자 있지만 우세
3. 실무 결정 시 기술 + 라이센스 + 시장 같이 고려, 면접에서 라이센스 답변 빠지면 깊이 부족 — 작은 회사는 H.264 또는 AV1, H.265는 협상 부담
